# Операторы сравнения 

• Равно
```
   if (a == b)
   {
       // Действия, если a равно b
   }
      
```

• Не равно
```
   if (a != b)
   {
       // Действия, если a не равно b
   }
```

• Больше
```
   if (a > b)
   {
       // Действия, если a больше b
   }
```

• Меньше
```
   if (a < b)
   {
       // Действия, если a меньше b
   }
```

• Больше или равно
```
   if (a >= b)
   {
       // Действия, если a больше или равно b
   }
```

• Меньше или равно
```
   if (a <= b)
   {
       // Действия, если a меньше или равно b
   }
```


# Логические операторы

• И
```
   if (a > 0 && b > 0)
   {
       // Оба числа положительные
   }
```

• ИЛИ
```
   if (a > 0 || b > 0)
   {
       // Хотя бы одно число положительное
   }
```

• НЕ
```
   if (!(a > 0))
   {
       // a не положительное
   }
```


# Операторы Арифметические

• Сложение
	`int sum = a + b;`

• Вычитание
	`int difference = a - b;`

• Умножение
	`int product = a * b;`

• Деление
	`int quotient = a / b; // Если a и b - целые числа`

• Остаток от деления
	`int remainder = a % b;`

• Унарные операторы
	`int positive = +a;`
	`int negative = -a;`
